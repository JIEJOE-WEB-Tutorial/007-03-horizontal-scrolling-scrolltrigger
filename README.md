## **Describe📋️**
当页面滑动到一定位置时，页面中的内容会固定到页面顶端，同时页面中的内容由竖向滑动变为横向滑动，<br>
该效果使用了GSAP的ScrollTrigger配合CSS的sticky粘性定位，相比传统的横向滑动效果、更加稳定，并且可以适配尺寸响应问题

<br>
